## App Review Information

**Demo Account:** Not applicable (no login required)

**Notes for Reviewer:**
PageFlow is a simple EPUB reader app. To test:
1. Launch the app
2. Tap the + button to import an EPUB file
3. Tap a book to start reading
4. Tap the screen to show/hide reading controls
5. Access settings via the gear icon

The app requires no account or internet connection. All books are stored locally on the device.
